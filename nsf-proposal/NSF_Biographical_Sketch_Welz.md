### Software and Digital Platforms (2023-Present)

**CopernicusAI Knowledge Engine** (2023-Present)
- Main Platform: https://www.copernicusai.fyi
- Hugging Face Space: https://huggingface.co/spaces/garywelz/copernicusai
- AI-powered research briefing and podcast generation system
- Operational platform with generated podcast episodes across multiple scientific disciplines
- Integration with multiple academic databases providing access to research papers
- Multi-voice AI podcast generation with RSS feed distribution
- Evidence-based content requiring minimum 3 research sources per briefing

**Programming Framework** (2023-Present)
- Hugging Face Space: https://huggingface.co/spaces/garywelz/programming_framework
- Zenodo: https://doi.org/10.5281/zenodo.18463442
- Universal method for process analysis using LLM-powered extraction and Mermaid visualization
- Domain-agnostic methodology demonstrated across multiple scientific disciplines
- Converts textual process descriptions into structured, computable diagrammatic representations

**Genome Logic Modeling Project (GLMP)** (2023-Present)
- Hugging Face Space: https://huggingface.co/spaces/garywelz/glmp
- First specialized application of Programming Framework to biological process visualization
- Biological processes mapped as interactive flowcharts
- JSON-based structured data in Google Cloud Storage
- Demonstrates feasibility of process visualization as computable research artifacts

**Research Paper Metadata Database** (2024-Present)
- Hugging Face Space: https://huggingface.co/spaces/garywelz/metadata_database
- GitHub Repository: https://github.com/garywelz/copernicusai-research-metadata
- Centralized metadata repository with AI-powered preprocessing
- Citation network analysis and structured JSON storage
- Foundation for unified multi-modal metadata representation

**Science Video Database** (2024-Present)
- Hugging Face Space: https://huggingface.co/spaces/garywelz/sciencevideodb
- Live Demo: https://scienceviddb-web-204731194849.us-central1.run.app/
- Curated searchable database of scientific video content
- Transcript-based search across multiple scientific disciplines
- Demonstrates multi-modal content integration for research
