## 3. Cloud Infrastructure Resources

### Google Cloud Platform (GCP)

**Services Utilized:**
- **Cloud Run:** Serverless container deployment for backend services
- **Firestore:** NoSQL database for metadata and application data
- **Cloud Storage:** Object storage for media files, transcripts, structured data, and research datasets
- **Cloud Functions:** Event-driven processing and automation
- **Vertex AI:** Enterprise-scale LLM deployment and orchestration
- **Cloud Tasks:** Background job queuing
- **Secret Manager:** Secure API key and credential management

**Current Usage:**
- Production CopernicusAI platform running on GCP
- Genome Logic Modeling Project (GLMP) data (biological processes) stored in Cloud Storage
- Research Paper Metadata Database infrastructure on GCP
- Science Video Database backend infrastructure

**Project Enhancements:**
- Expanded storage for unified metadata objects
- Increased compute capacity for enhanced processing and evaluation
- Additional Vertex AI usage for multi-model orchestration and reliability mechanisms
- Enhanced analytics infrastructure for user studies

**Cost Model:** Pay-as-you-go with budget allocation in project budget ($78,000 over 3 years)

**Adequacy:** Existing GCP infrastructure provides scalable, production-ready infrastructure for all project activities.

### Vercel (Frontend Hosting)

**Services Utilized:**
- Next.js application hosting
- CDN for static assets
- Serverless function hosting

**Current Usage:**
- Production CopernicusAI frontend hosted on Vercel
- Science Video Database frontend

**Cost Model:** Subscription-based with budget allocation in project budget ($3,000 over 3 years)
