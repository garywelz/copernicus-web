Wave Optics (Interference & Diffraction)

Physics Optics Complexity: high
← Back to Physics Database Programming Framework

Description

Wave optics explains interference and diffraction through the Huygens-Fresnel principle: each point on a wavefront emits secondary wavelets whose amplitudes superpose, and the path difference between contributions decides constructive versus destructive interference, producing the characteristic fringe intensity pattern set by wavelength and aperture geometry.
