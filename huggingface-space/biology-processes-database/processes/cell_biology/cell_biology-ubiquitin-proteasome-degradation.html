Ubiquitin-Proteasome Degradation

Biology Cell Biology Complexity: high
← Back to Biology Database Programming Framework

Description

Ubiquitin-proteasome system: an E1-E2-E3 enzyme cascade builds a K48-linked polyubiquitin chain on a substrate, which the 26S proteasome recognizes, deubiquitinates, unfolds, and degrades into peptides.
