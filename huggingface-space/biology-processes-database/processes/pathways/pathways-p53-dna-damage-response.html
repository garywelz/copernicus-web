p53 DNA Damage Response

Biology Pathways Complexity: high
← Back to Biology Database Programming Framework

Description

p53 DNA-damage response: ATM/ATR phosphorylate p53, disrupting MDM2 binding so p53 accumulates and induces p21-mediated arrest and repair; if damage is irreparable, p53 drives apoptosis, while re-synthesized MDM2 provides negative feedback.
