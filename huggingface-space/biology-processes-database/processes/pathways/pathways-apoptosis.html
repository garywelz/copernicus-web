Apoptosis Pathway

Biology Pathways Complexity: high
← Back to Biology Database Programming Framework

Description

Apoptosis: extrinsic death receptors activate caspase-8 while intrinsic stress is gated by the BCL-2 family at the mitochondrion; cytochrome c release forms the apoptosome and activates caspase-9, converging on executioner caspases that dismantle the cell.
