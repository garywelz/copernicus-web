Calvin Cycle (Light-Independent Reactions)

Biology Pathways Complexity: medium
← Back to Biology Database Programming Framework

Description

Calvin cycle: RuBisCO fixes CO2 onto ribulose-1,5-bisphosphate, the 3-phosphoglycerate product is reduced using ATP and NADPH to G3P, and most G3P is used to regenerate RuBP while a fraction is exported to make sugars.
