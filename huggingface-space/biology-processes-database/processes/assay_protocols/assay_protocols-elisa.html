ELISA

Biology Assay / Lab Protocols Complexity: high
← Back to Biology Database Programming Framework

Description

Sandwich enzyme-linked immunosorbent assay: a plate-bound capture antibody traps the analyte, an enzyme-conjugated detection antibody and substrate generate colour, and absorbance is read against a cutoff to call the result.
