Morphogen Gradient Patterning

Biology Development and Organismal Biology Complexity: high
← Back to Biology Database Programming Framework

Description

Morphogen gradient patterning: a localized source secretes a diffusible morphogen whose concentration gradient is read by cells against thresholds to switch on distinct target genes, with feedback shaping the gradient.
