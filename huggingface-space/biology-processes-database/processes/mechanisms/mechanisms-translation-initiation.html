Translation Initiation

Biology Mechanisms Complexity: medium
← Back to Biology Database Programming Framework

Description

Eukaryotic translation initiation: eIF4F binds the 5' cap, the 43S pre-initiation complex scans the 5' UTR for an AUG start codon, recognition triggers eIF2 GTP hydrolysis, and 60S joining forms an elongation-competent 80S ribosome.
