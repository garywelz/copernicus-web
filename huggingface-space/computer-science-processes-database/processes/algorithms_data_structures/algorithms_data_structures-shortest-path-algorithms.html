Shortest Path (Dijkstra)

Computer Science Algorithms & Data Structures Complexity: high
← Back to Computer Science Database Programming Framework

Description

Dijkstra's algorithm finds shortest paths from a source in a graph with non-negative weights: distances start at infinity except the source, and a priority queue repeatedly extracts the closest unfinalized vertex, finalizes it, and relaxes its outgoing edges, updating tentative distances until the queue is empty and the shortest-path tree is complete.
