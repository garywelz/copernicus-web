Topological Sort (Kahn's Algorithm)

Computer Science Algorithms & Data Structures Complexity: high
← Back to Computer Science Database Programming Framework

Description

Topological sorting linearly orders the vertices of a directed acyclic graph so every edge points forward: Kahn's algorithm computes in-degrees, repeatedly removes a zero in-degree vertex into the order while decrementing its neighbors, and enqueues newly freed vertices; if fewer than all vertices are output, the graph contained a cycle.
