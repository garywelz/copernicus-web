Minimum Spanning Tree (Kruskal / Prim)

Computer Science Algorithms & Data Structures Complexity: high
← Back to Computer Science Database Programming Framework

Description

A minimum spanning tree connects all vertices of a weighted graph with least total edge weight by a greedy rule: Kruskal sorts edges and adds the lightest that does not form a cycle (using union-find), while Prim grows a tree by repeatedly adding the lightest edge leaving it, until every vertex is connected.
