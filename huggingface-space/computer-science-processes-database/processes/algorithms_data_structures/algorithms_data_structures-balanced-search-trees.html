Balanced Search Trees (Rotations)

Computer Science Algorithms & Data Structures Complexity: high
← Back to Computer Science Database Programming Framework

Description

Self-balancing search trees keep operations at O(log n) by restoring a balance invariant after each insertion: a standard BST insert is followed by checking the height or color invariant, and when it is violated, single or double rotations and recoloring are applied up the path until the tree property is restored.
