Model Evaluation & Validation

Computer Science Machine Learning Complexity: high
← Back to Computer Science Database Programming Framework

Description

Model evaluation estimates generalization rather than training fit: metrics are chosen by task (precision/recall for classification, RMSE/R-squared for regression), k-fold cross-validation gives mean and variance of performance, the bias-variance regime is diagnosed, and a final held-out test confirms the model generalizes before acceptance.
