Virtual Memory Paging

Computer Science Systems & Architecture Complexity: high
← Back to Computer Science Database Programming Framework

Description

Paging translates a process's virtual address to a physical one: the address splits into a page number and offset, the TLB is consulted, and on a miss the page table is walked; if the page is not resident a page fault traps to the OS, which evicts a victim and loads the page from disk before forming the physical address.
