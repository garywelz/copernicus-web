NP-Completeness via Reduction

Computer Science Theoretical Computer Science Complexity: high
← Back to Computer Science Database Programming Framework

Description

To prove a problem NP-complete one shows it lies in NP and is NP-hard: membership is shown with a polynomial-time verifier, then a known NP-complete problem is reduced to it by a polynomial-time mapping that preserves yes-instances; verifying the reduction's size and correctness completes the proof.
