Routing Protocol Operation

Computer Science Computer Networks Complexity: high
← Back to Computer Science Database Programming Framework

Description

Routing protocols build forwarding tables by sharing reachability information: distance-vector protocols exchange distance tables with neighbors while link-state protocols flood link advertisements to build a full topology, each then computes shortest paths and recomputes routes whenever the topology changes, converging on a consistent routing table.
