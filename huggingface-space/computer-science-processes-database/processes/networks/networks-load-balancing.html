Load Balancing

Computer Science Computer Networks Complexity: high
← Back to Computer Science Database Programming Framework

Description

A load balancer spreads incoming requests across backend servers: it selects a target by an algorithm such as round-robin, least-connections, or weighted hashing, checks backend health and removes unhealthy nodes from the pool, then forwards the request, achieving an even and resilient distribution of load.
