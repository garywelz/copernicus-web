IP Addressing & Subnetting (CIDR)

Computer Science Computer Networks Complexity: high
← Back to Computer Science Database Programming Framework

Description

Subnetting divides an address block to fit host requirements: from the needed host count the required host bits and prefix length are computed, the block is split into subnets, and if a subnet is too small more host bits are borrowed; network and broadcast addresses are reserved and routes are aggregated with CIDR.
