Reliable Transfer with Sliding Window

Computer Science Computer Networks Complexity: high
← Back to Computer Science Database Programming Framework

Description

Sliding-window ARQ provides reliable, in-order delivery over an unreliable channel: the sender segments data and transmits a window of packets while running timers; arriving ACKs slide the window and admit new packets, while timeouts trigger retransmission of unacknowledged segments, continuing until all data is delivered.
