Buffer Overflow Exploit & Defense

Computer Science Security & Cryptography Complexity: high
← Back to Computer Science Database Programming Framework

Description

A buffer overflow occurs when unchecked input is copied past a buffer's bounds, overwriting the return address to redirect control flow to attacker code; modern defenses such as stack canaries, ASLR, and non-executable memory detect or block the attack, while proper bounds checking rejects the input outright.
