Intrusion Detection

Computer Science Security & Cryptography Complexity: high
← Back to Computer Science Database Programming Framework

Description

An intrusion detection system continuously watches network or host activity: events are collected and evaluated either by signature matching against known attacks or by comparison to an anomaly baseline; suspicious activity raises an alert, false positives feed back to tune the rules or model, and confirmed events trigger response.
