Role-Based Access Control (RBAC)

Computer Science Security & Cryptography Complexity: high
← Back to Computer Science Database Programming Framework

Description

RBAC grants permissions through roles rather than to users directly: an authenticated subject's assigned roles are activated in a session, and a requested operation is allowed only if some active role carries the permission and no separation-of-duty constraint is violated; every decision is logged for audit.
