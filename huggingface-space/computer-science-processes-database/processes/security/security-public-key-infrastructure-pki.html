Public Key Infrastructure (PKI)

Computer Science Security & Cryptography Complexity: high
← Back to Computer Science Database Programming Framework

Description

PKI binds identities to public keys through trusted certificate authorities: an entity generates a key pair and certificate signing request, the CA validates its identity and signs a certificate that chains to a trusted root; relying parties later verify the signature chain and revocation status before trusting it.
