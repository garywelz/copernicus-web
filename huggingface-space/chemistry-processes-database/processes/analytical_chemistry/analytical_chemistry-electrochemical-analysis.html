Potentiometry (Ion-Selective Electrode)

Chemistry Analytical Chemistry Complexity: high
← Back to Chemistry Database Programming Framework

Description

Potentiometry measures an ion's activity at zero current: a selective membrane develops a potential against a reference electrode following the Nernst/Nikolsky equation, and after calibration against standards the measured EMF is converted to concentration.
