Chromatographic Separation

Chemistry Spectroscopy Advanced Complexity: high
← Back to Chemistry Database Programming Framework

Description

Chromatography separates a mixture by differential partitioning between a mobile and a stationary phase: components with stronger stationary-phase affinity are retained longer; band broadening (van Deemter) limits resolution, which is tuned by adjusting phase and flow.
