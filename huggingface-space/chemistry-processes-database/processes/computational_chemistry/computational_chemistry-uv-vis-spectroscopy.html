QM/MM Multiscale Modeling

Chemistry Computational Chemistry Complexity: high
← Back to Chemistry Database Programming Framework

Description

QM/MM partitions a large system into a small quantum region treated with electronic structure and a classical force-field environment; link atoms handle the boundary and electrostatic embedding couples the regions, enabling reactions in enzymes or solution.
