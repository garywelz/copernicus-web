Nanoparticle Synthesis (LaMer)

Chemistry Materials Chemistry Complexity: high
← Back to Chemistry Database Programming Framework

Description

The LaMer model explains monodisperse nanoparticle synthesis: precursor decomposition raises monomer concentration until a critical supersaturation triggers a single burst of nucleation, which drops the concentration below threshold so no new nuclei form, and the existing nuclei grow by diffusion under capping-ligand control into uniform particles.
