Phase Diagram (Gibbs Phase Rule)

Chemistry Physical Chemistry Complexity: high
← Back to Chemistry Database Programming Framework

Description

A phase diagram is constructed from the Gibbs phase rule F = C - P + 2, which fixes the degrees of freedom: single-phase regions and the boundaries between them are mapped on temperature-composition axes, the lever rule gives phase amounts inside two-phase regions, and invariant points complete the diagram.
