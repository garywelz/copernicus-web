Description

HMM: states (hidden), observations (nucleotides/amino acids), transition P(s'|s), emission P(o|s). Profile HMM: Match, Insert, Delete states for multiple alignment. Viterbi: most likely state path given observations; DP recurrence. Forward-Backward: posterior P(state|obs); Forward/Backward algorithms. Baum-Welch: EM for parameter estimation. Applications: gene finding, protein families (HMMER), CpG islands, sequence alignment.

Source: Wikipedia; Durbin et al., Biological Sequence Analysis
