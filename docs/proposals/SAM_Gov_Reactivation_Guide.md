### Issue: Can't Remember Login Credentials

**Solution:**
- Use "Forgot Password" on SAM.gov login page
- Try both email addresses: garywelz@gmail.com and gwelz@jjay.cuny.edu
- May need to create new account if old one can't be recovered
- Contact SAM.gov help desk if needed

### Issue: DUNS Number Issues

**Solution:**
- Verify DUNS 017240183 is still active at https://www.dnb.com
- If DUNS is inactive, may need to reactivate with Dun & Bradstreet first
- Then use active DUNS in SAM.gov registration

### Issue: CAGE Code Issues

**Solution:**
- CAGE 8BWB1 should still be valid
- If issues, may need to contact Defense Logistics Agency (DLA)
- CAGE codes are typically permanent once assigned

### Issue: Business Type Classification

**Solution:**
- For SBIR, typically classified as "Small Business"
- Business structure: Individual/Sole Proprietor or LLC
- Verify correct classification for your entity type

## Important Notes

1. **Registration Must Be Active Before Submission**
   - DOE will verify SAM.gov registration
   - Inactive registration will cause proposal rejection

2. **UEI is Required**
   - Unique Entity Identifier (UEI) replaces DUNS for new awards
   - UEI is automatically assigned when registration is active
   - Include both DUNS and UEI in proposal

3. **Keep Information Current**
   - Update address, contact info, banking info as needed
   - Renew registration annually
   - Respond to any SAM.gov inquiries promptly

4. **Business Information**
   - Ensure business name matches exactly as used in proposals
   - Verify address is current
   - Keep contact information up to date
