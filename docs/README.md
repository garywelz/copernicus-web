# Documentation Index

This directory contains organized documentation for the CopernicusAI project.

## Directory Structure

### 📋 `planning/`
Planning documents, roadmaps, and strategic plans:
- Component improvement plans
- Refactoring plans
- Paper publication plans
- Endpoint extraction plans
- Descript integration plans

### 📊 `status/`
Status reports, deployment status, and test results:
- Deployment status updates
- Test results and test suite documentation
- Next steps and progress reports

### 🔌 `mcp/`
Model Context Protocol (MCP) server documentation:
- MCP server implementation plans
- Deployment and testing guides
- Proposal enhancements

### 📝 `proposals/`
Proposal-related administrative documents:
- SAM.gov registration status and guides
- Proposal validation checklists
- Administrative details

### 🎥 `video/`
Video production and feature documentation:
- Video feature roadmaps
- Video graphics integration analysis
- Video production prompts

### 🔧 `refactoring/`
Refactoring progress and status:
- Refactoring progress reports
- Refactoring reviews
- Extraction status updates

## Other Organized Directories

- **`../papers/`** - Research papers and drafts
- **`../scripts/`** - Utility shell scripts
- **`../tools/`** - External tools and JAR files
